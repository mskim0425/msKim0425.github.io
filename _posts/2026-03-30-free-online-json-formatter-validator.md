## FAQ

**Is my data safe?**
Yes. Everything runs client-side with JavaScript. Open your browser's Network tab to verify — zero outbound requests.

**Does it work offline?**
Once the page loads, yes. All logic runs in the browser.

**What's the max size?**
Comfortably handles files up to several megabytes. For 50MB+ files, use `jq` on the command line.

**Can it handle JSON with comments?**
Standard JSON doesn't support comments. The validator will flag them as errors, which is correct per the JSON spec.
