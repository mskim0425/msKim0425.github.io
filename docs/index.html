개인정보처리방침 (한국어)

최종 업데이트: 2026년 5월 13일

본 방침은 "쉬운 SNS"(이하 "이 앱")가 사용자의 개인정보를 어떻게 처리하는지 설명합니다. 이 앱은 사용자가 사진과 글을 Facebook과 X(Twitter)에 손쉽게 게시할 수 있도록 돕는 안드로이드 앱입니다.

1. 수집·이용하는 정보

항목출처이용 목적
Facebook 액세스 토큰Facebook 로그인(OAuth)사용자 본인의 Facebook에 게시
Facebook 사용자 이름Facebook 로그인 응답설정 화면에 "로그인된 계정" 표시
게시할 사진 및 글사용자가 앱 내에서 선택·입력Facebook 게시(또는 X 공유) 전송

2. 정보의 저장 위치

  • Facebook 액세스 토큰은 사용자의 기기 내에만 저장됩니다. 안드로이드의 EncryptedSharedPreferences로 암호화되어 보관됩니다.
  • 본 앱은 자체 서버를 운영하지 않으며, 위 정보를 외부 서버로 전송하지 않습니다.
  • 사진과 글은 게시 전송 직후 앱 메모리에서 폐기됩니다.

3. 제3자 제공

  • Meta (Facebook): 사용자가 게시 버튼을 누른 경우, 선택한 사진과 글이 Meta의 Graph API로 전송되어 사용자 본인의 Facebook에 게시됩니다. 이 처리에는 Meta의 데이터 정책이 적용됩니다.
  • X (Twitter): 본 앱은 X와 직접 API 연동을 하지 않습니다. 사용자가 X 게시를 선택한 경우 안드로이드의 표준 공유 인텐트를 통해 사진과 글이 X 공식 앱으로 전달되며, 그 이후의 처리는 X 공식 앱과 X의 정책이 담당합니다.
  • 위 두 경우를 제외하고, 본 앱은 어떠한 제3자에게도 사용자의 정보를 제공하지 않습니다.

4. 보관 기간

  • 액세스 토큰: 사용자가 설정 화면에서 "로그아웃"을 누르거나, 앱을 삭제하면 즉시 삭제됩니다.
  • 사진·글: 게시 시도가 끝나면 즉시 메모리에서 제거되며, 별도 저장하지 않습니다.

5. 권한 사용

  • INTERNET: Facebook Graph API 호출에 사용
  • READ_MEDIA_IMAGES (Android 13+) / READ_EXTERNAL_STORAGE (이전): 게시할 사진을 사용자가 직접 선택할 때만 사용. 갤러리를 자동으로 스캔하지 않습니다.

6. 사용자의 권리

  • 언제든 설정 화면에서 Facebook 로그아웃을 통해 저장된 토큰을 삭제할 수 있습니다.
  • 앱 삭제 시 저장된 모든 데이터가 함께 제거됩니다.
  • Facebook에 이미 게시된 글의 삭제는 Facebook 공식 앱/웹사이트에서 직접 수행하셔야 합니다.

7. 아동 개인정보

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 변경 사항

본 방침이 변경되는 경우 이 페이지의 "최종 업데이트" 날짜와 함께 갱신된 내용을 게시합니다. 중요한 변경 사항이 있을 경우 앱 실행 시 알려드립니다.

Privacy Policy (English)

Last updated: May 13, 2026

This policy explains how "Easy SNS" (the "App") handles your personal information. The App is an Android application that helps users post photos and text to Facebook and X (Twitter) with minimal steps.

1. Information We Collect and Use

ItemSourcePurpose
Facebook access tokenFacebook Login (OAuth)Post to your own Facebook account
Facebook display nameFacebook Login responseShow the signed-in account in Settings
Photos and text to postProvided by you in-appSend to Facebook Graph API (or share to X)

2. Where the Data Lives

  • The Facebook access token is stored only on your device, encrypted via Android's EncryptedSharedPreferences.
  • The App does not operate any backend server. None of the above information is sent to a server we control.
  • Photos and text are discarded from app memory immediately after the post attempt completes.

3. Third Parties

  • Meta (Facebook): When you tap Post, the selected photos and text are sent to Meta's Graph API and published to your own Facebook account. This processing is governed by Meta's Data Policy.
  • X (Twitter): The App does not integrate with X's API directly. When you choose to share to X, the photos and text are handed to the official X app through Android's standard share intent. Any further handling is performed by the X app under its own policy.
  • The App does not share your information with any other third parties.

4. Retention

  • Access token: removed immediately when you tap "Log out" in Settings, or when you uninstall the App.
  • Photos and text: cleared from memory as soon as the post attempt finishes; nothing is persisted.

5. Permissions

  • INTERNET: used to call the Facebook Graph API.
  • READ_MEDIA_IMAGES (Android 13+) / READ_EXTERNAL_STORAGE (older): used only when you explicitly pick a photo. The App does not scan your gallery in the background.

6. Your Rights

  • You can delete the stored token at any time by tapping "Log out" in Settings.
  • Uninstalling the App removes all locally stored data.
  • To delete content already posted to Facebook, please use the official Facebook app or website.

7. Children's Privacy

The App is not directed to children under 14 and does not knowingly collect personal information from children.

8. Changes

If this policy changes we will update this page along with the "Last updated" date. Material changes will be surfaced in the App on next launch.
