사용자 데이터 삭제 안내 (한국어)

최종 업데이트: 2026년 5월 13일

이 페이지는 "쉬운 SNS"(이하 "이 앱")에서 사용자가 본인의 데이터를 어떻게 삭제할 수 있는지 안내합니다. 이 앱은 사용자 데이터를 외부 서버에 저장하지 않으므로, 모든 데이터는 사용자가 직접 본인의 기기와 페이스북 계정에서 제거할 수 있습니다.

요약: 이 앱은 자체 서버를 운영하지 않습니다. 모든 데이터는 (1) 사용자의 안드로이드 기기 내부 또는 (2) 사용자 본인의 페이스북 계정에만 존재합니다. 따라서 데이터 삭제를 위해 저희에게 별도 요청을 보내실 필요가 없습니다.

1. 이 앱이 다루는 데이터 (재확인)

  • Facebook 액세스 토큰 — 사용자의 안드로이드 기기 내부에 암호화되어 저장됨 (외부 전송 없음).
  • Facebook 사용자/페이지 이름 — 설정 화면 표시용. 기기 내부에만 저장.
  • 게시할 사진/글 — 사용자가 직접 선택·입력. 게시가 끝나면 앱 메모리에서 즉시 폐기.

자세한 내용은 개인정보처리방침을 참고해 주세요.

2. 앱 안에서 직접 삭제하기 (가장 간단한 방법)

  1. "쉬운 SNS" 앱 실행
  2. 홈 화면 우상단 설정(⚙️) 아이콘 누르기
  3. 페이스북 로그아웃 버튼 누르기
  4. 확인 다이얼로그에서 "네, 로그아웃" 누르기

이 절차로 기기 내부에 저장된 Facebook 액세스 토큰과 사용자 이름이 즉시 삭제됩니다.

3. 앱 자체를 삭제하기 (완전 제거)

  1. 안드로이드 홈 화면에서 "쉬운 SNS" 아이콘 길게 누르기
  2. 앱 정보 또는 제거 선택
  3. 삭제 확인

앱이 제거되면 안드로이드가 자동으로 앱이 저장한 모든 로컬 데이터(토큰, 캐시 등)를 함께 삭제합니다.

4. 페이스북 측에서 앱 권한 해제하기

위 1번·2번 절차만으로도 이 앱은 더 이상 페이스북 데이터에 접근할 수 없습니다. 다만 페이스북 계정 쪽에 남아 있는 "이 앱이 한 번이라도 연결됐었다"는 기록까지 지우시려면:

  1. 페이스북 공식 앱 또는 facebook.com 설정 페이지 접속
  2. 설정 및 개인정보 보호 → 설정 → 비즈니스 통합 (또는 앱 및 웹사이트) 이동
  3. 목록에서 "쉬운 SNS" 또는 해당 앱 찾기
  4. 삭제(Remove) 클릭

이 단계까지 마치시면 Meta(Facebook) 측에 남아 있던 본 앱의 권한 부여 기록도 완전히 제거됩니다.

5. 페이스북에 이미 게시된 글

이 앱을 통해 페이스북에 게시된 사진과 글은 사용자 본인의 페이스북 계정에 일반 게시물로 남아 있습니다. 이를 삭제하시려면 페이스북 공식 앱 또는 웹사이트에서 해당 게시물을 찾아 직접 삭제해 주세요. 이 앱은 게시 이후 해당 글에 더 이상 관여하지 않습니다.

6. 추가 도움이 필요하신 경우

위 절차로 해결되지 않는 데이터 삭제 요청이 있으시면 아래로 연락해 주십시오. 영업일 기준 30일 이내에 답변드립니다.

  • 본인 페이스북 계정 식별 정보 (이름 또는 이메일)
  • 앱 사용 추정 시점
  • 삭제를 원하시는 데이터의 종류

User Data Deletion Instructions (English)

Last updated: May 13, 2026

This page explains how a user of "Easy SNS" (the "App") can delete their own data. The App does not store user data on any backend server, so all data can be removed directly by the user from their own device and Facebook account.

Summary: The App operates no backend server. All data lives either (1) on the user's Android device or (2) inside the user's own Facebook account. You do not need to send us a request to have data deleted.

1. Data Handled by the App (Recap)

  • Facebook access token — stored encrypted on the user's Android device (never uploaded).
  • Facebook user / page name — used in the Settings screen, stored on-device only.
  • Photos and text to post — selected/typed by the user, discarded from app memory right after the post attempt completes.

For details see the Privacy Policy.

2. Delete from within the App (Easiest)

  1. Launch the "Easy SNS" app.
  2. Tap the Settings (⚙️) icon at the top-right of the home screen.
  3. Tap Log out of Facebook.
  4. Confirm "Yes, log out" in the dialog.

This immediately removes the Facebook access token and user name stored on your device.

3. Uninstall the App (Full Wipe)

  1. Long-press the "Easy SNS" icon on your Android home screen.
  2. Choose App info or Uninstall.
  3. Confirm.

Android will remove all local data the App stored (tokens, caches) along with the app.

4. Revoke App Permission on Facebook

Steps 2–3 are already enough to stop the App from accessing your Facebook. If you also want to remove the "this app was once connected" record kept on Facebook's side:

  1. Open the official Facebook app, or facebook.com settings.
  2. Go to Settings & privacy → Settings → Business Integrations (or Apps and Websites).
  3. Find "Easy SNS" in the list.
  4. Click Remove.

After this, the authorization record on Meta's side is fully cleared.

5. Posts Already Published on Facebook

Photos and text published via the App live in your Facebook account as regular posts. To delete them, go to the official Facebook app or website, find the post, and delete it there. The App has no further connection to those posts after publication.

6. Need Further Help?

If you have a deletion request that the steps above do not solve, please contact us. We respond within 30 business days.
Email: sothoughtful@deali.net
Suggested subject: [Easy SNS] Data deletion request

Including the following information in your email speeds up the process:

  • An identifier for your Facebook account (name or email)
  • Approximate dates when you used the App
  • The kinds of data you want deleted
